
# standard version add boostrap to Rails 

```sh
yarn add bootstrap@next jquery @popperjs/core
```

In gemfile add
```
gem 'bootstrap', '~> 4.6.0'
```

run
```shell
bundle
```


in `/app/assets/stylesheets/application.css` 

add 
```
 *= require bootstrap
```

rename to application.scss
